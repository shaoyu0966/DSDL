# Pipelined Multiplier

### Feature

- `mult_fast`: 4-bit multiplier with hierarchy design

  

### Usage

- Test if the multiplier functions correctly

  ```bash
  iverilog -o lab1.vpp lab1.v
  vvp lab1.vpp
  ```

- Show waveform

  1. Open GTKwave
  2. *Open New Tab* > `lab2.vcd`
  3. *Read Save File* > `lab2.sav`
